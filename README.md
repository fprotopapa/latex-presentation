# latex-presentation
Presentation made with Tex for shorter talks.

With:
- Titlepage
- Table of content
- Bibliography
- Easy and clean slide template
- With notes

Notes can be used togheter with [pdfpc](https://github.com/pdfpc/pdfpc), e.g.

```
 pdfpc --notes=right -s presentation.pdf
 ```

 Compiled with:
 - lualatex
 - pdflatex
